# Vibes ✨

_Your daily dose of cosmic guidance._

---

## What is Vibes?

Vibes is your personal space for spiritual insights. Pull a tarot card, discover your numbers, read your horoscope—all in one beautifully calm place.

No accounts. No noise. Just vibes.

---

## What You'll Find

**Tarot Readings**  
Three-card spreads for clarity. One reading per day—make it count.

**Numerology**  
Your Life Path and Destiny numbers, decoded from your birth date.

**Daily Horoscopes**  
Personalized predictions based on your zodiac sign.

**Birth Charts**  
Visual wheel display of your complete birth chart with sun, moon, and rising signs.

**Moon Phases**  
Current moon phase, rituals, and zodiac influences.

**Chinese Zodiac**  
Your Chinese zodiac sign with traits and compatibility.

**Compatibility**  
Love compatibility readings between zodiac signs.

**Daily Affirmations**  
Personalized affirmations with cosmic themes.

**Your Profile, Saved**  
Your birthday and preferences stay with you. Come back anytime.

---

## Advanced Features

**Multiple Profiles**  
Create and switch between profiles for yourself, friends, or family.

**Data Export/Import**  
Backup and restore your profile and readings.

**Share as Image**  
Download any reading as a beautiful PNG image to share.

**PWA Support**  
Add to your home screen for offline access on mobile.

---

## The Vibe

Vibes is designed to feel like a breath of fresh air. Dark mode that's actually dark. Smooth animations. No clutter. No ads. No account required.

Just you and the cosmos.

---

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Package Manager:** Bun
- **Styling:** Tailwind CSS
- **Components:** shadcn/ui (Radix)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **State:** React Hooks + localStorage

---

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun dev

# Build for production
bun run build

# Run type check
bun run type-check
```
